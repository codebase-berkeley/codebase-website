This Semester's Projects
Each semester CodeBase works in teams of 8-9 to build a project for 3 months. Check out this semester's projects below.
Visumenu
Chatbot
Visumenu is a company that indexes telephone menus so customers can easily select the service option they need. We'll be implementing a solution specific to UC Berkeley’s phone menus and building a Facebook Messenger chatbot for users to interact with the data.
YiTuuX
Cloud Computing
YiTuuX develops artificial intelligence and machine learning based tools to enable healthcare workers. We'll be creating a user-facing API that is customizable, can manage heavy traffic, and handles requests to YiTuuX’s AI software.
ResultCare
App Development
ResultCare is a platform that provides clinicians with evidence and cost based information to help improve patient care. Project developers will integrate social network features with the ResultCare platform and retrofit ResultCare’s mobile apps with a new wireframe.
Riffyn
Data Analytics
Riffyn provides research design and analytics software to R&D organizations. We'll develop python modules and applications for statistical analysis and visualization of Riffyn users’ data.
Mentored Project
App Development
Mentored project members will be learn the basics of software engineering and web application development through building an admissions management app for the club.
