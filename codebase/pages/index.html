This Semester's Projects
Each semester CodeBase works in teams of 8-9 to build a project for 3 months. Check out this semester's projects below.
VisuMenu
Chatbot & Backend
VisuMenu is a search engine for automated phone menus. We’re providing a backend implementation for UC Berkeley’s phone systems and building a Facebook chatbot for users to interact with the menus.
YiTuuX
Cloud Computing
YiTuuX is creating AI-driven healthcare solutions. We’re building backend solutions to help YiTuux scale and manage the deployment of their Diabetes detection algorithm.
ResultCare
App Development & Web Backend
ResultCare is developing a data platform for medical professionals. We’re upgrading their mobile apps with new designs and social networking features.
Riffyn
Data Analytics
Riffyn creates collaborative software for biotech and chemical industries. We’re developing data analytics and visualization software for Riffyn’s users.
Mentored Project
App Development
Mentored project members will be learn the basics of software engineering and web application development through building an admissions management app for the club.
